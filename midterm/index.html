IMA is a BFA at NYU Tisch. We use code (and other technologies) to make art (instead of paints, etc)!
Our program is about learning the technologies of the future, while other programs tend to be more traditional.
I call myself a designer but everyone at IMA defines themselves differently!
I have been really interested in embodiment in the context of interaction design. A lot of my work also explores computer vision machine learning models.
